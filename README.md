# Pong Arcade Game Using Python

## Project Overview
This project is a classic Pong Arcade Game developed using Python and the Turtle graphics module. The game simulates a two-player table tennis game where players control paddles to hit the ball and score points.

The project demonstrates object-oriented programming, game logic, and graphical interface design.

---

## Technologies Used
- Python
- Turtle Graphics
- Object-Oriented Programming (OOP)

---

## Game Features
- Two-player gameplay
- Keyboard-controlled paddles
- Real-time ball movement
- Score tracking system
- Collision detection
- Smooth gameplay interface

---

## Controls

Player 1 (Left Paddle):
- Move Up: W
- Move Down: S

Player 2 (Right Paddle):
- Move Up: Up Arrow
- Move Down: Down Arrow

---

## Game Rules
- Each time a player misses the ball, the opponent scores a point.
- First player to reach the winning score wins.
- The ball speed increases gradually during gameplay.
